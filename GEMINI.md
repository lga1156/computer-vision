# Лабораторные работы по компьютерному зрению

## Общие правила

- В качестве основного источника информации для решения задач используй только материалы лекций (в формате .txt), которые расположены в директории ./public
- Если в материалах лекций не хватает информации для решения задачи, то используй внешние источники информации, но обязательно укажи их в коде
- Название готового .ipynb файла должно быть 'hwN.ipynb', где N - номер лабораторной работы
- Готовый .ipynb файл экспортируй в .html файл с названием 'Meleshenko-hwN.html', где N - номер лабораторной работы
- Все файлы лабораторных работ должны быть сохранены в директории ./labs/hwN, где N - номер лабораторной работы
- Все файлы индивидуальных заданий должны быть сохранены в директории ./labs/taskN, где N - номер лабораторной работы

## Технологический стек

- Используй Python 3.x
- Для работы с изображениями строго OpenCV (cv2) и NumPy
- Для нейросетей — PyTorch
- Для графиков — Matplotlib
- Для работы с файлами — os

## Coding Style

- Не оставляй комментарии в .ipynb файлах
- Выности загрузку всех библиотек в первый блок .ipynb файла
- Код и визуализация должны быть в отдельных блоках

## Экспорт в .html файл

Для экспорта .ipynb в .html используй команду терминала: jupyter nbconvert --to html ./labs/hwN/hwN.ipynb --output Meleshenko-hwN.html
